Documentation XAMPP:
1. Installer XAMPP sur l'ordinateur
2. Une fois installé, lancer XAMPP et activer le module Apache en cliquant sur "start" corespondant.
3. Dans la partie droite du logiciel, cliquer sur "Explorer", ce qui nous emmène vers les
sous dossiers de XAMPP.
4. Chercher le dossier "htdocs" et y déposer le dossier de son portrait chinois
5. Enfin, dans votre navigateur tappez l'adresse suivant : http://localhost/PORTRAIT_CHINOIS_CLARA/
